# gjPDFs Updated on the App Store

[**gjPDFs**](/projects/gjPDFs) has been updated to **v2026.4** on the [App Store](https://apps.apple.com/app/gjpdfs/id6760037275).

This is a maintenance release with the following changes:
- Fixed an issue with the app's window minimising when opening another document via Finder.
- Fixed the zoom/scale not being correctly applied when changing pages.
- Fixed an issue with zoom in/out not using the correct scale.
