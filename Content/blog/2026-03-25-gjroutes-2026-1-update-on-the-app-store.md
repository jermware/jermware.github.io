# gjRoutes Updated on the App Store

[**gjRoutes**](/projects/gjRoutes) has been updated to **v2026.1** on the [App Store](https://apps.apple.com/app/gjroutes/id6742377844).

This is an enhancement release with the following changes:
- iOS - **gjRoutes** is now included in Open With menu and Share Sheet for PDF files.
- macOS - **gjRoutes** is now included in Open With menu for PDF files.
- Strava activity list now shows distance.
- Elevation chart now displays a message when elevation data are not available.
